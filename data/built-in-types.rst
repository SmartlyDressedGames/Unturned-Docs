C# Built-in Types

   :widths: 20 20 40 20
   :header-rows: 1
   
   * - C# type

     - Range
     - Default Value

     - ``true`` or ``false``

   * - int8

   * - uint8

     - ``0`` to ``255``
     - ``0``
   * - int16
     - short
     - ``-32768`` to ``32767``
     - ``0``
   * - uint16

     - ``0``
