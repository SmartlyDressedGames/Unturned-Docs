Filter Assets
=============

Radiation filters can be used to replenish the quality of radiation-proof :ref:`masks <doc_item_asset_mask>`.

This inherits the :ref:`ItemAsset <doc_item_asset_intro>` class.

Item Asset Properties
---------------------

**GUID** *32-digit hexadecimal*: Refer to :ref:`GUID <doc_data_guid>` documentation.

**Type** *enum* (``Filter``)

**Useable** *enum* (``Filter``)

**ID** *uint16*: Must be a unique identifier.

Filter Asset Properties
-----------------------

Radiation filters have no unique asset properties. Refer to parent classes for additional properties.
